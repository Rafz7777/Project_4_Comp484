CSUN Map Location Game
Overview
----------------------
The CSUN Map Location Game is an interactive web application built with the Google Maps API. The game challenges users to locate specific CSUN campus buildings by clicking on a map. It demonstrates JavaScript event handling, DOM manipulation, and working with an external API.

Features
--------
Interactive Map Game

Users are prompted to find 5 locations (4 chosen by the user, 1 assigned).

Double-click on the map to make a guess.

Correct guesses highlight the location in green; incorrect guesses show the correct location in red.

Each guess has a 50-meter tolerance for easier selection.

Map Customization
-----------------
Terrain map centered on the target area.

Labels hidden for a clean, campus-like view.

Zoom and panning disabled to focus gameplay.

User Interface
--------------
Status messages, answer history, and score displayed to the left of the map.

Correct/incorrect answers are color-coded.

Timer tracks completion time.

High score recorded using localStorage.

Reset button restarts the game.

Extra Features
--------------
Pulsing animation effect for selected locations.

Final score summary displayed at the end.

Technologies Used
-----------------
HTML5, CSS3

JavaScript (ES6) with jQuery

Google Maps JavaScript API
